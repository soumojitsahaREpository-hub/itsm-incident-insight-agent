# Dataset Source

The raw CSV file at data/raw/incident_event_log_dataset.csv is included in this repository for reproducibility, learning, and transparency. It is the source input used to create the processed dataset at data/processed/incidents_level_df.csv.

The local DuckDB database at data/processed/itsm_agent.duckdb is generated from the processed CSV during local setup and should not be committed.

This project uses the Kaggle dataset titled "Incident Event Log Dataset" by uploader mithilkotawadekar:

- Dataset URL: https://www.kaggle.com/datasets/mithilkotawadekar/incident-event-log-dataset

Users should review the Kaggle dataset page for the latest license and usage terms.

This repository does not claim ownership of the dataset and does not present it as live production data.
